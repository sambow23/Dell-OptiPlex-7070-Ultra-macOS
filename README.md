## Hardware compatibility

#### What works
- CPU power management
- GPU acceleration and video codecs
- SSD (SATA/NVME)
- Wi-Fi (itlwm)
- All USB ports
- Sleep
- USB-C Video
- Ethernet
- Bluetooth (with AirPods patch)

#### Not Working
- Audio Output (support not planned)
- iCloud Services (support not planned)
